### Notes
The main objective of this project is to implement `MVVM` and `BLoC` principles while ensuring `clean code` practices.

---
#### MVVM Structure
```
lib/
├── view/
│   ├── widgets/
├── viewmodel/
├── model/
│   ├── entities/
│   ├── repositories/
│   ├── data_providers/
│   ├── models/
├── constants.dart
└── main.dart
```
